Install in Minutes

Start in 3 Easy Steps

No technical experience needed - just copy and paste the commands

Bash Script
# Quick Installation (Recommended) bash <(curl -fsSL https://vpsplus.dev/install.sh) # Or using wget bash <(wget -qO- https://vpsplus.dev/install.sh) # Install specific protocols only bash <(curl -fsSL https://vpsplus.dev/install.sh) --protocols=xray,wireguard,ssh # Direct GitHub installation git clone https://github.com/vpsplus/vpsplus-script.git && cd vpsplus-script && bash install.sh
1

Update Your System

Make sure your Ubuntu 20.04+, Debian 11+, or CentOS 8+ system is up to date

2

Run Installation Command

Copy the command above, paste it in your terminal (SSH), and press Enter

3

Access Control Panel

After completion, type vpsplus or menu to access the interactive dashboard

Management Commands
# Open main control panel vpsplus # Show all services status vpsplus --status # Add new SSH user vpsplus --add-user # Setup Xray (VLESS/VMess/Trojan) vpsplus --xray # Setup WireGuard VPN vpsplus --wireguard # Setup L2TP/IPsec vpsplus --l2tp # Setup OpenVPN vpsplus --openvpn # Monitor system resources vpsplus --monitor # Update script to latest version vpsplus --update
