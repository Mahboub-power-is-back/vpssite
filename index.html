التثبيت خلال دقائق

ابدأ في 3 خطوات فقط

لا تحتاج لأي خبرة تقنية، فقط نسخ ولصق الأوامر

Bash Script
# التثبيت السريع (الطريقة الموصى بها) bash <(curl -fsSL https://vpsplus.dev/install.sh) # أو باستخدام wget bash <(wget -qO- https://vpsplus.dev/install.sh) # للتثبيت المباشر من GitHub git clone https://github.com/vpsplus/vpsplus-script.git && cd vpsplus-script && bash install.sh
1

تحديث النظام

تأكد من تحديث نظام Ubuntu 20.04+ أو Debian 11+ الخاص بك

2

تشغيل أمر التثبيت

انسخ الأمر أعلاه والصقه في الطرفية (SSH) ثم اضغط Enter

3

الوصول للوحة التحكم

بعد اكتمال التثبيت، اكتب vpsplus أو menu للوصول للوحة التحكم

Commands
# فتح لوحة التحكم الرئيسية vpsplus # عرض حالة جميع الخدمات vpsplus --status # إضافة مستخدم SSH جديد vpsplus --add-user # إعدادات Xray (VLESS/VMess) vpsplus --xray # مراقبة الإحصائيات vpsplus --monitor # تحديث السكريبت لأحدث إصدار vpsplus --update
