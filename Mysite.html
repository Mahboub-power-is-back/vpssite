Quick Installation

Copy and paste the following command to install the script on your VPS. Requirements: Ubuntu/Debian system with root access.

Installation Command 
# Update system packages
apt update && apt upgrade -y

# Download and run the installation script
wget https://raw.githubusercontent.com/vps-script-manager/main/install.sh

# Make the script executable
chmod +x install.sh

# Run the installation
./install.sh

# Follow the on-screen instructions to configure your proxy services

Important Notes:

  • Make sure you have a fresh Ubuntu/Debian VPS (recommended: Ubuntu 20.04+)
  • Run the script as root user or with sudo privileges
  • The script will automatically configure firewall (UFW/iptables)
  • After installation, you can access the management menu with: vps-menu
